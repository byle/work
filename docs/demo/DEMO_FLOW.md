# 演示账号与测试流程

## 演示账号

- 管理员：`admin / admin123`
- 调度：`dispatcher / dispatcher123`
- 现场：`site01 / site123`

## 演示地址

- 后台：`http://localhost:8080`
- 移动端：`http://localhost:8081`
- API：`http://localhost:3000`

## 推荐演示顺序

### 1. 登录后台

使用管理员或调度账号登录后台。

### 2. 查看统计看板

进入 `统计看板`：

- 查看项目数量
- 查看工单状态数量
- 查看清单数量

### 3. 创建项目模板

进入 `项目模板`：

- 新建一个项目模板
- 配置默认工单模板

### 4. 创建项目

进入 `项目列表`：

- 新建项目
- 设置负责人和项目成员
- 选择项目模板

### 5. 创建 / 指派工单

进入 `工单列表`：

- 查看模板自动生成的工单
- 指派现场执行人
- 设置审核人

### 6. 创建搭建清单

进入 `清单列表`：

- 创建搭建清单
- 录入一条明细
- 导入 CSV 文本或文件
- 导出 CSV

### 7. 现场执行

使用现场账号登录移动端：

- 查看“我的工单”
- 进入工单详情
- 开始执行
- 上传现场附件
- 提交审核并填写备注

### 8. 审核流转

使用调度或审核角色：

- 查看工单详情
- 查看审计日志
- 审核通过或驳回

## 推荐测试点

- 不同角色权限是否生效
- 指派后移动端是否能看到工单
- 附件是否可上传、访问、删除
- 审核备注是否落入日志与工单描述
- 一键启停脚本是否正常
