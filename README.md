# 舞台租赁工单系统

面向舞台租赁公司的项目协同与现场执行系统，覆盖项目创建、工单流转、模板化搭建清单、现场执行反馈等核心业务场景，支持 `Web` 后台与手机端 `H5/小程序` 轻量操作。

## 项目目标

为舞台租赁业务建立一套统一的工单系统，解决以下问题：

- 项目、工单、搭建清单分散在线下表格和聊天记录中，难以统一管理
- 搭建/撤场任务依赖人工传达，执行状态不透明
- 现场人员需要使用手机快速查看任务、勾选完成、上传反馈
- 常用活动类型的搭建内容重复率高，需要模板化复用
- 搭建清单存在手工录入和表格导入需求，需要统一结构化沉淀

## 适用业务场景

- 舞台、灯光、音响、LED 屏、桁架、背景板等租赁项目执行
- 活动项目的进场、彩排、正式执行、撤场全过程管理
- 现场搭建清单、物料清单、执行清单的创建、导入与分发
- 项目经理、调度、仓库、搭建人员、现场负责人协同作业

## 核心功能规划

### 1. 项目管理

- 创建项目，维护项目名称、地点、活动时间、进撤场时间、彩排时间等信息
- 记录客户信息、负责人、现场联系人、备注说明
- 以项目为主线统一归集工单、搭建清单、执行记录和附件

### 2. 工单管理

- 基于项目创建工单，例如搭建工单、运输工单、撤场工单、补料工单
- 支持工单状态流转：待分配、进行中、已完成、已关闭
- 支持按角色分派给项目经理、现场负责人、搭建人员等执行人
- 支持现场回传图片、备注、异常说明和完成时间

### 3. 模板化创建

- 支持预设标准项目模板和工单模板
- 支持按活动类型快速生成常用搭建项
- 支持模板版本管理，便于后续迭代优化
- 支持从模板一键生成项目搭建清单和配套工单

### 4. 搭建清单管理

结合你提供的“搭建清单”样式，系统需要支持以下能力：

- 手工录入搭建清单明细
- 从 Excel 表格导入搭建清单
- 支持清单表头信息维护：项目名称、项目地点、活动时间、进场时间、撤场时间、彩排时间
- 支持清单明细字段：序号、内容、规格、数量、单位、备注
- 支持分组展示，例如外场物料清单、大屏与舞台设备等分类
- 支持生成现场查看版本，方便手机端快速浏览与勾选执行

### 5. 现场轻量操作端

手机端建议支持 `H5`，后续可扩展到 `小程序`：

- 查看本人待处理项目与工单
- 查看搭建清单和任务明细
- 勾选已完成项
- 提交现场照片、异常、补充备注
- 快速反馈缺料、损坏、临时变更
- 支持弱培训成本，界面尽量简单直观

### 6. Web 后台管理

- 项目、工单、模板、清单、人员、角色统一管理
- 支持表格导入导出
- 支持工单分配、进度追踪、异常处理
- 支持项目维度查看搭建进度与执行情况
- 支持后续扩展统计报表

## 建议角色设计

- 超级管理员：系统配置、人员与权限管理
- 项目经理：创建项目、生成工单、管理搭建清单
- 调度/内勤：派工、协调进撤场安排
- 现场负责人：跟进执行、处理异常
- 搭建人员：接收任务、反馈完成情况
- 仓库/物料人员：确认设备物料出入库与备料状态

## 建议数据模型

首期建议围绕以下核心实体设计：

- 项目（Project）
- 工单（WorkOrder）
- 工单模板（WorkOrderTemplate）
- 项目模板（ProjectTemplate）
- 搭建清单（SetupList）
- 搭建清单明细（SetupListItem）
- 人员/用户（User）
- 角色/权限（Role / Permission）
- 附件（Attachment）
- 执行日志（OperationLog）

## 技术方案建议

### 客户端

- `Web` 后台：适合项目管理、模板维护、清单录入、导入导出、统计查询
- 手机端：优先采用 `H5` 快速落地，后续根据使用频率再扩展 `微信小程序`

### 服务端

- 提供统一 API 服务，供 Web 后台和手机端共用
- 支持认证鉴权、文件上传、模板生成、Excel 导入解析、工单状态流转

### 数据库

可选：

- `MySQL`：生态成熟，团队更容易上手
- `PostgreSQL`：适合更复杂的数据查询、JSON 扩展和后续统计分析

如果没有既定技术偏好，首期建议优先选择 `PostgreSQL`，后续在模板配置、复杂查询、日志审计等方面会更灵活。

## 典型业务流程

1. 创建项目
2. 选择项目模板或手工录入项目信息
3. 生成搭建清单与相关工单
4. 分配给项目经理、现场负责人、搭建人员
5. 现场人员通过手机端查看清单并执行
6. 执行过程中回传照片、备注、异常信息
7. 完成搭建、彩排保障、正式执行、撤场
8. 项目结束后沉淀模板与复盘数据

## 开发优先级建议

### 第一阶段（MVP）

- 用户登录与角色权限
- 项目创建与管理
- 工单创建与分配
- 搭建清单录入
- Excel 导入搭建清单
- 手机端查看清单与工单
- 现场勾选完成与备注反馈

### 第二阶段

- 项目模板与工单模板
- 图片上传与异常上报
- 工单流转记录
- 导出打印
- 基础数据统计

### 第三阶段

- 小程序版本
- 物料/设备联动
- 仓库管理联动
- 成本与利润统计
- 多项目排期与资源冲突预警
