# 项目骨架说明

## 目录

```text
.
├── apps
│   ├── admin-web
│   └── mobile-h5
├── docs
├── server
└── sql
```

## 技术选型

- `server`：`Node.js + Express + TypeScript`
- `apps/admin-web`：`React + Vite + TypeScript`
- `apps/mobile-h5`：`React + Vite + TypeScript`

## 后续建议

- 为 `server` 增加模块化目录和数据库连接层
- 为 `admin-web` 增加路由、布局和 API 封装
- 为 `mobile-h5` 增加移动端页面导航和状态管理
