# 发布说明

## 项目名称

舞台租赁工单系统

## 当前版本能力

### 1. 用户与权限

- 支持管理员、调度、现场三类角色
- 支持登录、鉴权、接口权限控制
- 支持后台用户管理

### 2. 项目管理

- 支持项目创建
- 支持项目负责人、项目成员
- 支持项目模板
- 支持按模板自动生成工单

### 3. 工单管理

- 支持工单创建
- 支持执行人、审核人配置
- 支持工单指派
- 支持工单状态流转：
  - `pending_assign`
  - `in_progress`
  - `pending_review`
  - `approved`
  - `rejected`

### 4. 搭建清单

- 支持搭建清单创建
- 支持清单明细录入、编辑、删除
- 支持 CSV 文本导入
- 支持 CSV 文件导入
- 支持 CSV 导出

### 5. 移动端现场执行

- 支持移动端登录
- 支持我的工单查看
- 支持开始执行、提交审核、审核通过 / 驳回
- 支持备注填写
- 支持现场附件上传、删除、访问

### 6. 运营与审计

- 支持统计看板
- 支持项目 / 工单搜索和状态筛选
- 支持项目 / 工单详情查看
- 支持审计日志记录与查看
- 支持数据库备份脚本
- 支持 Docker 一键启停

## 默认账号

- 管理员：`admin / admin123`
- 调度：`dispatcher / dispatcher123`
- 现场：`site01 / site123`

## 访问地址

- 后台：`http://localhost:8080`
- 移动端：`http://localhost:8081`
- 后端：`http://localhost:3000`

## 发布方式

```bash
cp .env.example .env
./scripts/start.sh
```

## 备份方式

```bash
./scripts/backup-db.sh
```
