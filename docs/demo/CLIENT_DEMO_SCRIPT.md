# 客户演示话术（简版）

## 1. 系统定位

这是一套面向舞台租赁公司的项目执行与协同系统，主要解决：

- 项目创建和模板复用
- 工单分派和执行跟踪
- 搭建清单录入、导入、导出
- 现场移动端轻量操作
- 审核留痕与附件管理

## 2. 演示路径

### 第一步：看板

向客户展示：

- 项目总数
- 工单总数
- 待处理工单
- 清单数量

### 第二步：项目模板与项目创建

说明：

- 常规活动项目可以复用模板
- 项目创建后能自动生成标准工单

### 第三步：工单协同

展示：

- 调度设置执行人和审核人
- 工单状态变化清晰可追踪

### 第四步：清单管理

展示：

- 清单明细录入
- 通过 CSV 快速导入
- 清单可导出给现场或供应链使用

### 第五步：现场执行

展示移动端：

- 登录后查看我的工单
- 开始执行
- 上传现场附件
- 填写备注并提交审核

### 第六步：审核与留痕

展示：

- 审核通过 / 驳回
- 审计日志记录谁在什么时间做了什么操作

## 3. 可扩展方向

可继续扩展：

- 微信小程序版本
- Excel 导入
- 统计报表
- 审批流增强
- 多仓库 / 设备资产管理
